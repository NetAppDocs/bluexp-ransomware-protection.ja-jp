= BlueXP  ランサムウェア対策機能へのロールベースアクセス
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
BlueXP  ランサムウェア対策では、特定の機能やアクションに対する各ユーザのアクセス権をロールで管理します。

BlueXP  ランサムウェア対策では、次のロールが使用されます。

* 組織管理者
* ビューア（組織管理者以外）
